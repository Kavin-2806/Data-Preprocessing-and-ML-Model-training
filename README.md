# Streamlit App

This repository contains a Streamlit dashboard for end-to-end data preprocessing, exploratory data analysis (EDA), model training, explainability, and clustering.

## Features

- Data preprocessing: missing values, encoding, scaling
- EDA: plots, correlation analysis, PCA, outlier detection
- Machine learning: model comparison, AutoML, hyperparameter tuning, feature selection
- Pipeline-based preprocessing with training and prediction support
- SHAP explainability with optional runtime optimization
- KMeans clustering and downloadable results
- Experiment tracking and download-ready model artifact

## Usage

```bash
streamlit run project.py
```
